# WWW '26 Build Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** A mobile-first, offline-first PWA remake of the Elsewhere '26 festival guide for Jonas + girlfriend, with an AI "For You" ranking and calendar export.

**Architecture:** One-shot build pipeline extracts the 677-event array from the source JS bundle, AI-scores every event against the couple's taste, and bakes a static `public/events.json`. A vanilla-JS PWA reads that file, does all filtering/favourites/ICS in-browser, and precaches everything via a service worker for zero-signal use on-site.

**Tech Stack:** Node (build scripts, ESM), vanilla HTML/CSS/JS (no framework), service worker PWA, Claude Workflow for the AI scan, Google Calendar MCP (optional push).

**Time constraint:** Festival likely starts Tue Jul 7. Target: installable on phone ASAP. Prioritise a working offline browser first; AI scan + calendar can be layered.
